# PianoSprout App

Verwandeln Sie Ihr Gerät in ein magisches Klavier! Keine Vorkenntnisse erforderlich. Tippen Sie einfach auf die hervorgehobenen Tasten, und unsere intelligente Engine kümmert sich um den Rhythmus. Lustig, einfach und entspannend für jeden.

Installieren Sie die App im [App Store](https://apps.apple.com/app/1519974306)

Feedback bitte über ein [Issue](https://github.com/embbnux/pianosprout-app/issues/new). **Geben Sie keine sensiblen Daten im Issue an**.

## Beschreibung

Spielen Sie wunderschöne Musik in Sekunden – Keine Erfahrung nötig!

PianoSprout ist nicht nur ein Simulator; es ist ein lustiges Musikspielzeug, das jeden wie einen Profi klingen lässt. Ob Sie ein Kind sind oder jung geblieben, Sie werden lieben, wie einfach es ist, Ihre Lieblingsmelodien zu spielen.

【Was macht es so spaßig?】

- Magischer Modus (Automatische Dauer)

Vergessen Sie komplexes Timing! Tippen Sie einfach auf die richtige Taste, wenn sie aufleuchtet.

Die App spielt die Note automatisch für die perfekte Dauer. Sie kontrollieren die Geschwindigkeit, wir übernehmen den Rhythmus. Es fühlt sich an wie Magie!

- Coole Sounds entdecken

Gelangweilt vom Standard-Klavier? Wechseln Sie sofort zu Gitarre, Geige oder anderen Instrumenten.

Hochwertige Soundeffekte (SoundFonts) machen jedes Tippen zu einem Genuss.

- Einfach & Entspannend

Leucht-Guide: Folgen Sie den Hinweisen, um klassische Lieder wie "Jingle Bells" mühelos zu spielen.

Kreative Freiheit: Wechseln Sie in den freien Spielmodus und genießen Sie die Multi-Touch-Tastatur. Verbinden Sie sich mit Ihrem inneren Musiker ohne den Stress von Unterrichtsstunden.

【Funktionen】

Farbenfrohe und saubere Benutzeroberfläche.

Funktioniert hervorragend auf iPhone und iPad (probieren Sie die Dual-Tastatur!).

Perfekt für eine schnelle kreative Pause oder um die Kleinen zu unterhalten.
