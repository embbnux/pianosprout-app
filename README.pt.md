# PianoSprout App

Transforme seu dispositivo em um piano mágico! Nenhuma habilidade necessária. Basta tocar nas teclas destacadas e nosso motor inteligente cuida do ritmo para você. Divertido, simples e relaxante para todos.

Instale o aplicativo na [App Store](https://apps.apple.com/app/1519974306)

Dê feedback criando um [issue](https://github.com/embbnux/pianosprout-app/issues/new). **Não envie dados sensíveis no issue**.



## Descrição

Toque Músicas Lindas em Segundos – Nenhuma Experiência Necessária!

PianoSprout não é apenas um simulador; é um brinquedo musical divertido que faz qualquer um soar como um profissional. Seja você uma criança ou apenas jovem de coração, vai adorar como é fácil tocar suas músicas favoritas.

【O que o torna divertido?】

- Modo Mágico (Duração Automática)

Esqueça o timing complexo! Basta tocar na tecla correta quando ela acender.

O aplicativo toca automaticamente a nota pela duração perfeita. Você controla a velocidade, nós cuidamos do ritmo. Parece mágica!

- Explore Sons Legais

Entediado com o piano padrão? Mude para Violão, Violino ou outros instrumentos instantaneamente.

Efeitos sonoros de alta qualidade (SoundFonts) tornam cada toque satisfatório.

- Simples e Relaxante

Guia Iluminado: Siga as dicas para tocar músicas clássicas como "Jingle Bells" sem esforço.

Liberdade Criativa: Mude para o modo de toque livre e aproveite o teclado multitoque. Conecte-se com seu músico interior sem o estresse das aulas.

【Recursos】

Interface colorida e limpa.

Funciona muito bem no iPhone e iPad (experimente o Teclado Duplo!).

Perfeito para uma pausa criativa rápida ou para entreter os pequenos.
