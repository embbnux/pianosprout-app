# PianoSprout App

기기를 마법의 피아노로 변신시키세요! 기술이 필요 없습니다. 강조 표시된 건반을 탭하기만 하면 스마트 엔진이 리듬을 처리해 줍니다. 모두를 위한 재미있고 간단하며 편안한 앱입니다.

[App Store](https://apps.apple.com/app/1519974306) 에서 설치하세요.

피드백은 [issue](https://github.com/embbnux/pianosprout-app/issues/new) 를 생성해 주세요. **issue에 민감한 데이터를 포함하지 마세요**.

## 설명

몇 초 만에 아름다운 음악 연주 – 경험 불필요!

PianoSprout 는 단순한 시뮬레이터가 아닙니다. 누구나 프로처럼 연주할 수 있게 해주는 재미있는 음악 장난감입니다. 어린이든 동심을 가진 어른이든, 좋아하는 곡을 연주하는 것이 얼마나 쉬운지 마음에 드실 겁니다.

【무엇이 재미있나요?】

- 매직 모드 (자동 지속)

복잡한 타이밍은 잊으세요! 건반에 불이 들어올 때 올바른 건반을 누르기만 하면 됩니다.

앱이 자동으로 완벽한 길이로 음을 연주합니다. 속도는 당신이 조절하고, 리듬은 우리가 처리합니다. 마법 같아요!

- 멋진 사운드 탐색

일반 피아노가 지루하신가요? 기타, 바이올린 또는 다른 악기로 즉시 전환하세요.

고품질 사운드 효과(SoundFonts)로 탭할 때마다 만족스럽습니다.

- 간단하고 편안함

라이트 업 가이드: 힌트를 따라 "징글벨" 같은 명곡을 힘들이지 않고 연주하세요.

창의적 자유: 자유 연주 모드로 전환하여 멀티터치 키보드를 즐기세요. 레슨 스트레스 없이 내면의 음악적 감성을 깨워보세요.

【특징】

다채롭고 깔끔한 인터페이스.

iPhone 및 iPad 에서 훌륭하게 작동합니다 (듀얼 키보드를 사용해 보세요!).

빠른 창의적 휴식이나 아이들을 즐겁게 해주기에 완벽합니다.
