# PianoSprout App

將您的設備變成神奇鋼琴！無需技巧。只需點擊突出顯示的琴鍵，我們的智能引擎將為您處理節奏。簡單有趣，適合所有人放鬆。

在 [App Store](https://apps.apple.com/app/1519974306) 下載安裝

如有反饋，請創建 [issue](https://github.com/embbnux/pianosprout-app/issues/new)。**請勿在 issue / 討論中提交敏感數據**。



## 應用描述

幾秒鐘就能彈奏美妙音樂 – 無需經驗！

PianoSprout 不僅僅是一個模擬器；它是一個有趣的音樂玩具，讓任何人聽起來都像專業人士。無論您是孩子還是童心未泯，您都會愛上彈奏喜愛曲調的輕鬆感覺。

【有什麼好玩的？】

- 魔法模式（自動時長）

忘記複雜的計時！只要在琴鍵亮起時點擊正確的琴鍵即可。

應用程序會自動以完美的時長播放音符。您控制速度，我們處理節奏。感覺像魔法一樣！

- 探索酷炫聲音

厭倦了標準鋼琴？瞬間切換到吉他、小提琴或其他樂器。

高品質音效 (SoundFonts) 讓每一次點擊都令人滿足。

- 簡單 & 放鬆

亮燈指引：跟隨提示輕鬆彈奏像“鈴兒響叮當”這樣的經典歌曲。

創意自由：切換到自由演奏模式，享受多點觸控鍵盤。無需課程壓力，與內心的音樂家連接。

【功能特性】

色彩豐富且整潔的界面。

完美支持 iPhone 和 iPad（試試雙鍵盤！）。

非常適合快速創意小憩或娛樂小朋友。
