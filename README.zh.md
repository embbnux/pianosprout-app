# PianoSprout App

将您的设备变成神奇钢琴！无需技巧。只需点击突出显示的琴键，我们的智能引擎将为您处理节奏。简单有趣，适合所有人放松。

在 [App Store](https://apps.apple.com/app/1519974306) 下载安装

如有反馈，请创建 [issue](https://github.com/embbnux/pianosprout-app/issues/new)。**请勿在 issue / 讨论中提交敏感数据**。

## 应用描述

几秒钟就能弹奏美妙音乐 – 无需经验！

PianoSprout 不仅仅是一个模拟器；它是一个有趣的音乐玩具，让任何人听起来都像专业人士。无论您是孩子还是童心未泯，您都会爱上弹奏喜爱曲调的轻松感觉。

【有什么好玩的？】

- 魔法模式（自动时长）

忘记复杂的计时！只要在琴键亮起时点击正确的琴键即可。

应用程序会自动以完美的时长播放音符。您控制速度，我们要处理节奏。感觉像魔法一样！

- 探索酷炫声音

厌倦了标准钢琴？瞬间切换到吉他、小提琴或其他乐器。

高品质音效 (SoundFonts) 让每一次点击都令人满足。

- 简单 & 放松

亮灯指引：跟随提示轻松弹奏像“铃儿响叮当”这样的经典歌曲。

创意自由：切换到自由演奏模式，享受多点触控键盘。无需课程压力，与内心的音乐家连接。

【功能特性】

色彩丰富且整洁的界面。

完美支持 iPhone 和 iPad（试试双键盘！）。

非常适合快速创意小憩或娱乐小朋友。
